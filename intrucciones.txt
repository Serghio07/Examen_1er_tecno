-----Intrucciones------
Para comenzar a trabajar con la tienda en línea de perfumes usando **XAMPP**, es esencial seguir los siguientes pasos iniciales para configurar el entorno local y acceder al proyecto. A continuación se presenta un instructivo detallado dividido en cuatro párrafos, que incluye la activación de XAMPP y la colocación del archivo correcto en la carpeta adecuada.

 <<<<<<<<<Activar XAMPP>>>>>>>>>>>  
Primero, asegúrate de tener **XAMPP** instalado en tu computadora. XAMPP es un software que incluye Apache (servidor web), MySQL (base de datos), y PHP (lenguaje de programación), que son necesarios para ejecutar tu proyecto en local. Abre XAMPP y en su panel de control, activa los servicios de **Apache** y **MySQL** haciendo clic en el botón de "Start" junto a cada uno de ellos. Esto inicia el servidor web local (Apache) y el servicio de base de datos (MySQL).

<<<<<<<<<<<<Colocar el proyecto en la carpeta correcta>>>>>>>>>>>>**  
Una vez que XAMPP esté activo, debes colocar tu proyecto en la carpeta adecuada. Localiza la carpeta donde está instalado XAMPP (normalmente en *C:\xampp* en Windows) y navega hasta la subcarpeta llamada **htdocs** (*C:\xampp\htdocs*). Dentro de esta carpeta, crea una nueva carpeta llamada **EXamen** y copia todos los archivos y subcarpetas de tu proyecto web (incluyendo archivos PHP, CSS, JavaScript, y las imágenes necesarias) en esta nueva carpeta. Esto permitirá que tu proyecto sea accesible desde el servidor local.
 <<<<<<<<<<<<<Acceder a tu proyecto desde el navegador>>>>>>>>>>>>>>>>>>**  
Ahora que tu servidor y base de datos están activos, y tus archivos están en la carpeta correcta, puedes acceder a tu sitio web. Abre un navegador web y escribe la siguiente URL en la barra de direcciones: `http://localhost/EXamen/index.php`. Esto cargará el archivo `index.php` desde la carpeta **EXamen** que acabas de crear en **htdocs**. Si todo está configurado correctamente, la página de inicio de tu tienda de perfumes debería aparecer, como la que se muestra en las imágenes anteriores.

<<<<<<<<<<<Verificar la conexión con la base de datos>>>>>>>  
Finalmente, asegúrate de que tu sitio web esté correctamente conectado a la base de datos. Para esto, utiliza **phpMyAdmin**, una herramienta incluida en XAMPP que te permite gestionar bases de datos MySQL de manera gráfica. Accede a **phpMyAdmin** escribiendo `http://localhost/phpmyadmin` en tu navegador. Aquí, debes crear una base de datos llamada **tienda_perfumes** y asegurarte de que las tablas necesarias (como `perfumes`, `carrito`, `usuarios`, `mensajes_contacto`) estén correctamente configuradas. Tu archivo PHP debe contener el código para conectarse a esta base de datos, y podrás verificar si la conexión funciona correctamente accediendo a las distintas funcionalidades del sitio web (añadir productos al carrito, enviar mensajes, etc.).


Para construir una tienda de perfumes en línea como la que se muestra en las imágenes, se debe seguir una serie de pasos que involucran tanto la estructura de la base de datos como la creación de interfaces gráficas que permitan al usuario interactuar con el sistema de manera eficiente. A continuación, se proporciona un instructivo dividido en cuatro párrafos que resume los pasos fundamentales para lograrlo.

En primer lugar, es necesario diseñar la estructura de la base de datos que almacenará la información. De acuerdo con el esquema presentado, la base de datos incluye al menos cuatro tablas: `perfumes`, `carrito`, `mensajes_contacto` y `usuarios`. La tabla `perfumes` debe contener los detalles de cada producto, tales como el nombre, precio y descripción. La tabla `carrito` almacenará los productos seleccionados por el usuario, mientras que `mensajes_contacto` manejará las solicitudes o preguntas que los clientes envíen a través del formulario de contacto. Por último, `usuarios` gestionará la información básica de cada cliente.

En segundo lugar, se debe desarrollar la interfaz de usuario utilizando HTML, CSS y PHP (u otro lenguaje adecuado para el lado del servidor). La página de inicio incluirá botones de navegación como “Inicio”, “Perfumes”, “Pedidos” y “Contactos”, para facilitar el desplazamiento entre secciones. Las tarjetas de productos (perfumes) deben mostrar el nombre, precio y descripción breve del producto, con botones para añadir al carrito o ver más detalles. Esto debe estar vinculado dinámicamente a la base de datos para que el contenido se actualice automáticamente.

El tercer paso consiste en crear la funcionalidad del carrito de compras. Esto implica que los productos seleccionados se reflejen en el carrito del usuario, como se muestra en la tercera imagen. La lógica del carrito debe manejar el cálculo de los subtotales y el total general, permitiendo a los usuarios ajustar las cantidades de cada producto y eliminar elementos si es necesario. Asimismo, al hacer clic en "Realizar mi compra", debe procesarse la información, ya sea para generar un pedido o integrar con una pasarela de pagos.

Por último, es esencial implementar un sistema de contacto funcional. La sección de contacto debe tener un formulario donde los usuarios puedan enviar su nombre, correo electrónico y mensaje. Esta información se almacenará en la tabla `mensajes_contacto` de la base de datos para ser revisada posteriormente por los administradores del sistema. Este formulario debe incluir validaciones básicas para asegurarse de que el usuario introduzca la información requerida correctamente. Con estas instrucciones básicas, se puede crear un sistema de tienda en línea completamente funcional y dinámico.
